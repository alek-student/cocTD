# Castle of Cards TD

Prosta gra z gatunku tower defense. Wyróżnia się tym, że zamiast ustalonej ścieżki, jest ona generowana na podstawie przeszkód i wież stawianych przez gracza.
Jak nazwa gry sugeruje, przeszkody są w formie kart, które gracz dobiera losowo na początku gry i pomiędzy falami.

# Sterowanie

Karty lub wieże można kliknąć żeby zacząć je przeciągać i ponownie aby je postawić.
Żeby anulować przeciąganie wystarczy wcisnąć prawy przycisk myszy.
Trzymane karty można obracać klawiszem R.
